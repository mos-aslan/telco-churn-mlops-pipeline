Step 02 - Initial EDA and Churn Relationship Analysis

This folder contains the outputs generated during the exploratory data analysis phase.

Subfolders:
- tables: CSV summary tables generated from the raw Telco Churn dataset.
- figures: PNG visualizations generated during EDA.

Main contents:
- Missing values summary
- Numerical descriptive statistics
- Target distribution
- Churn rate by categorical features
- Numerical feature summaries grouped by Churn Value
- Distribution plots and boxplots

These files are used later in the final project report and documentation.
